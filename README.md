# Projeto Faculdade - Ciência de Dados

Este repositório foi criado com o objetivo de organizar estudos, atividades acadêmicas e pequenos projetos desenvolvidos durante minha graduação em Ciência de Dados.

A proposta é utilizar este espaço como uma forma de acompanhar minha evolução no aprendizado, reunindo conteúdos práticos, exercícios e experiências adquiridas ao longo do curso.

## Objetivos do repositório

- Registrar conhecimentos aprendidos durante a graduação
- Praticar programação e análise de dados
- Desenvolver projetos simples para aprendizado contínuo
- Organizar materiais e conteúdos estudados
- Aprimorar gradualmente minhas habilidades na área de tecnologia

## Áreas de interesse

- Python
- SQL
- Ciência de Dados
- Análise de Dados
- Machine Learning
- Automação

## Observações

Atualmente estou em processo de aprendizado e desenvolvimento acadêmico, utilizando este repositório principalmente para prática, organização dos estudos e construção gradual de experiência na área.
